# ArogyaSahayak AI Agents

## 1. Prescription Analysis Agent

### Purpose

Analyzes uploaded prescription images and extracts important medical information.

### Responsibilities

* Read prescription images
* Extract medicine names
* Extract dosage instructions
* Identify precautions
* Generate patient-friendly explanations
* Generate caregiver summaries

### Input

* Prescription image

### Output

* Patient summary
* Medicine list
* Dosage schedule
* Precautions
* Caregiver guidance

---

## 2. Bill Analysis Agent

### Purpose

Analyzes hospital bills and explains charges in simple language.

### Responsibilities

* Read bill images and PDFs
* Extract cost categories
* Identify expensive components
* Generate simplified explanations

### Input

* Hospital bill image or PDF

### Output

* Bill summary
* Cost breakdown
* Expensive charges
* Simple explanation

---

## 3. Action Plan Agent

### Purpose

Generates patient and caregiver action plans from prescription data.

### Responsibilities

* Create reminders
* Generate caregiver tasks
* Suggest next steps
* Provide follow-up guidance

### Input

* Prescription analysis output

### Output

* Reminders
* Caregiver actions
* Next steps
* Support note

---

## 4. Language Translation Agent

### Purpose

Makes healthcare information understandable in regional languages.

### Responsibilities

* Translate healthcare instructions
* Simplify medical language
* Improve accessibility

### Input

* Medical text
* Target language

### Output

* Translated text
* Simplified explanation
