# Contributing to ArogyaSahayak AI

Thank you for contributing to ArogyaSahayak AI.

## Development Workflow

1. Fork or clone the repository.
2. Create a feature branch.
3. Make changes.
4. Test your changes.
5. Commit using meaningful commit messages.
6. Submit a merge request.

## Coding Standards

* Follow Python best practices.
* Use meaningful variable names.
* Add documentation for major features.
* Avoid committing secrets or API keys.

## Reporting Issues

Create an issue with:

* Description
* Steps to reproduce
* Expected behavior
* Screenshots if applicable
