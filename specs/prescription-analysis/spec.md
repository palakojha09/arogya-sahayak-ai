# Prescription Analysis

## Objective
Analyze prescription images and generate patient-friendly healthcare guidance.

## User Story
As a patient,
I want to upload a prescription image,
So that I can understand my medicines and dosage schedule.

## Inputs
- Prescription image (JPG, PNG)

## Outputs
- Patient summary
- Medicine list
- Dosage schedule
- Precautions
- Caregiver summary

## Acceptance Criteria
- Prescription image uploads successfully
- Medicine details are extracted
- Output is returned as structured JSON
- Errors are handled gracefully
