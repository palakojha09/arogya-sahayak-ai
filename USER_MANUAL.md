# ArogyaSahayak AI User Manual

## Features

* Prescription Analysis
* Hospital Bill Analysis
* Action Plan Generation
* Medical Translation

## Prescription Analysis

1. Upload prescription image.
2. Click Analyze.
3. View medicines, dosage schedule, and caregiver guidance.

## Bill Analysis

1. Upload bill image or PDF.
2. Click Analyze Bill.
3. Review cost breakdown and explanations.

## Action Plan

1. Generate action plan from prescription analysis.
2. Review reminders and caregiver actions.

## Translation

1. Enter healthcare text.
2. Select target language.
3. View translated output.
